# README

## Login JWT API

### Users
* GET /api/v1/users
* GET /api/v1/users/:id
* POST /api/v1/users
* DELETE /api/v1/users/:id
* PATCH /api/v1/users/:id

### Posts
* GET /api/v1/posts
* GET /api/v1/posts/:id
* POST /api/v1/posts
* DELETE /api/v1/posts/:id
* PATCH /api/v1/posts/:id

### Database creation 
* rails db:migrate && rails db:seed
